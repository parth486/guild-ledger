Lightweight admin tool to log and track contact interactions with live search, filters, and CSV export.

== Description ==
Urbana Guild Ledger lets your team capture contact interactions with structured fields (contact name, company/council, date, interaction type, notes, and lead status). The admin list supports live search and filters, and you can export entries to CSV.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/urbana-guild-ledger` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The Urbana menu will appear in the admin sidebar with the Guild Ledger subpage.
